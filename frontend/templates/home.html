A Visual Representation of the Internet

The choropleth map represents the proportion of http vs https websites found in the dataset grouped by country. As an example, in the US 45.49% of the websites use secure communications (https) while the remaining 54% of websites use the insecure http protocol.
A caveat of this picture is that while nearly half of the internet traffic is served through http, websites that don't do online transactions are perfectly fine delivering static content via the http protocol.



Technical Note: We used part of the massive dataset (20TB) made public by the Common Crawl Project project, which was provided for December 2019. Later joining this dataset with the GeoLite database to get a geo-reference of each website crawled by the CommonCrawl. This product includes GeoLite2 data created by MaxMind, available from https://www.maxmind.com
